# BrainBloom Kids Privacy Policy

This repository is intended for hosting the public BrainBloom Kids Privacy Policy using GitHub Pages.

After enabling GitHub Pages from the `main` branch and `/root`, the policy should be available at:

```text
https://<github-username>.github.io/<repo-name>/privacy-policy/
```

Example:

```text
https://sjavagar.github.io/brainbloom-privacy/privacy-policy/
```

Files:

- `index.html` - redirects to the privacy policy
- `privacy-policy/index.html` - standalone privacy policy page with embedded logo and CSS

Do not add Play Console Account ID or signing files to this public repository.
